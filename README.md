# Super Granny Racing 2 – Luxe Edition 🛒
## Overview 📌

Super Granny Racing 2 – Luxe Edition is a multiplayer prototype developed in Unreal Engine 5.6 as part of a learning exercise focused on online systems and gameplay replication.

The project explores the idea of a chaotic multiplayer “grocery racing game”, where players take control of competitive grandmothers pushing shopping carts through a supermarket arena. The objective is simple: collect items in a specific order as fast as possible and reach the checkout before the other players.

While originally designed as a prototype, the game emphasizes fun, emergent multiplayer interactions and physics-driven gameplay chaos.

## Core Concept 🧓 

Instead of traditional racing tracks, players compete inside a supermarket environment.

Each player:
- Controls a grandmother character with a shopping cart
- Follows a randomized shopping list
- Must collect items in the correct order
- Can physically interact with other players
- Can push, bump, and disrupt opponents using their cart
The goal is to complete the shopping list and reach the checkout first.

## Gameplay Features 🎮
- Multiplayer supermarket racing gameplay
- Item collection system based on ordered shopping lists
- Physical cart-based movement with intentionally “goofy” handling
- Player interaction through bumping and item throwing
- Third-person 3C system (Character, Camera, Control)
- Win condition based on completing shopping list + reaching checkout

## Technical Features 🧠
This prototype was mainly developed as a multiplayer systems learning exercise in Unreal Engine, focusing on:
- Full multiplayer replication (RPC, RepNotify)
- Steam Online Subsystem integration
- Online sessions management
- Replicated character animations
- Physics-based interactions between carts
- IK systems for character animation consistency in multiplayer
- Networked gameplay logic for item collection and progression

## Gameplay Feel 🧱
The gameplay is intentionally:
- Slightly clunky
- Chaotic
- Physics-driven
- Comedy-oriented
The goal was not realism, but fun multiplayer interactions and emergent situations.

## Technical Notes 🛠️
- Built in Unreal Engine 5.6
- PC platform only
- Multiplayer focused prototype
- Not actively developed beyond the prototype phase
- Designed as a learning project for online systems and replication workflows

## Status ⚠️
This project is a short-term prototype (2-week development cycle) created for educational purposes.
It is no longer actively developed, but serves as a demonstration of:
- Multiplayer implementation in Unreal Engine
- Gameplay replication systems
- Steam session integration
- Physics-based interaction design
